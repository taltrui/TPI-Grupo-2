Puntuación: 0

Instrucciones del juego

En este juego didáctico, el usuario será presentado con una imagen grande a la izquierda, que representará una situación determinada, y a su derecha se presentarán dos imagenes más chicas que serviran como opción 1 y 2.
El usuario dispondrá de un límite de tiempo de 30 segundos para completar el cuestionario de 5 preguntas.
Si el usuario logra acumular al menos 3 puntos, gana el juego.
Mucha suerte!
