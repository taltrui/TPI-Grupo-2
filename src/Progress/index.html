Avances

Sobre el proyecto

Este proyecto de educación vial se desarrolló utilizando jQuery, HTML y CSS, con un diseño responsive que se adapta a diferentes dispositivos. Se implementó un pequeño router con jQuery para manejar la navegación interna de la página, proporcionando una experiencia de usuario fluida y dinámica.

Público objetivo

El público objetivo del proyecto son niños y adolescentes, con el propósito de enseñar y concienciar sobre la seguridad vial. A través de actividades interactivas y educativas, se busca inculcar buenas prácticas viales desde una edad temprana.

Identidad visual

La identidad visual del proyecto se centra en el uso de colores verde y azul/celeste. Estos colores no solo aportan frescura y tranquilidad, sino que también están asociados con la seguridad y la confianza, elementos clave en la educación vial.

Github y Figma

Para el desarrollo y mantenimiento del proyecto, se utilizó GitHub Pages para hostear la página web. El código se guarda y versiona en GitHub utilizando Git. Figma fue la herramienta elegida para diseñar las interfaces tanto en versiones móviles como de escritorio.
